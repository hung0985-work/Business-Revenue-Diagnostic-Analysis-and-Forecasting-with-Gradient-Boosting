# Business-Revenue-Diagnostic-Analysis-and-Forecasting-with-Gradient-Boosting
Phân tích chẩn đoán tình hình kinh doanh của doanh nghiệp TMĐT thời trang, xác định nguyên nhân sụt giảm doanh thu từ khách hàng, danh mục sản phẩm và tồn kho, đồng thời xây dựng mô hình Gradient Boosting để dự báo Revenue và COGS hỗ trợ ra quyết định.
